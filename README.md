# Digital Image Processing Scripts

This repository contains Python scripts developed as part of a Digital Image Processing course. It includes implementations of various image processing, filtering, segmentation, and compression techniques.

## Features

- Image transformations such as grayscale, negative, logarithmic, and gamma corrections
- Histogram processing and enhancement
- Noise generation and reduction
- Filtering techniques for smoothing and edge detection
- Morphological operations
- Thresholding methods
- Edge detection and segmentation
- Image compression algorithms
- Advanced techniques including k-means clustering, bit plane slicing, and gray level slicing
